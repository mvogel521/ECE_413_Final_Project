Project Description
The Heart Track project is an IoT-enabled web application for tracking heart rate and blood oxygen saturation levels. It combines a hardware sensor device and web platform, allowing users to configure the frequency and time of day for measurements. The IoT device prompts users to take readings, stores data locally if disconnected, and transmits it to the web application upon reconnection. The web app, built with Node.js, Express, and MongoDB, enables users to create accounts, manage devices, and view their health data through daily and weekly summaries, with data visualized on responsive charts.
